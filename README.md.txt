# 🏠 House Price Prediction

This project uses **Linear Regression** to predict house prices based on features like 
Living Area, Bedrooms, Bathrooms, Quality, and Year Built.  

## 📂 Project Structure
- `house_price.ipynb` → Jupyter Notebook with code
- `mini_house.csv` → Dataset used
- `house_price_model.pkl` → Saved trained model
- `requirements.txt` → Dependencies
- `README.md` → Project documentation
